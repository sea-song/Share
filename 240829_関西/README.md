###  （特别推荐）本地观光推荐：滋贺县大津市琵琶湖（日本最大淡水湖）
京都人的后花园，从京都出发电车20~40分钟，观光客主要为日本人。我几乎每年黄金周都会去一次。

**但是好不好玩得看天气好不好**

- 琵琶湖大津会馆（びわ湖大津館）
