# 注意事项
## 携带物品清单
- 一次性毛巾x3
    - 白驹庄x1
    - 露营地x2
- 沐浴露洗发水旅行小瓶装+牙刷牙膏旅行装
- 一次性内裤x6（长野）~x10（全行程）
- 装衣服的软袋
    - 装脏衣服的袋子/塑料袋
- 装鞋子的袋子
    - 拖鞋，徒步鞋
- 带上适合这个气温的户外活动服装（8月）：另外有两个晚上温度可能低至15度（室内活动），准备好一件适合的**室内**舒适御寒衣物
    - 非高原：白天跟广州差不多热，晚上23~25度左右，比较凉快
    - 白驹庄标高2100米的气温：白天18~25度，夜晚最低12度（室内15度~）
    - 北八岳坪庭标高2200米的气温：白天18~25度，夜晚不在此活动
    - 雾峰标高1500~1950米的气温：白天18~25度，夜晚最低12度（室内15度~）
    - 露营地：白天可能热至30度，晚上18~20度左右
- （可选）徒步装备：徒步鞋，户外背包，水壶
- 带耳塞：长野第一晚大通铺带耳塞会比较好睡眠(自带或者我可以提供)
